### Installing

```
npm install @huehuejs/name-convention --save
```

### Usage

```
const nameConverter = new NameConventionConverter(Gluers, Splitters); 
```

| Gluers         | Splitters |
| -------------- | ----------|
| LowerCamelCase | AnyCase   |
| UpperCamelCase | CamelCase |
| LowerSnakeCase | SnakeCase |
| UpperSnakeCase | KebabCase |
| LowerKebabCase |           |
| UpperKebabCase |           |
